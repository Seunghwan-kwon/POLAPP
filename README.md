# POLAPP

POLAPP은 광운대학교 소프트웨어학부 졸업 작품 관련 프로그램으로, 대한민국 경찰 관련 어플리케이션이다.

## Project Structure

- `lib/main.dart`: App entry point
- `lib/pages/`: Main UI screens
- `lib/services/`: Shared business logic and service layer
- `assets/`: Images and other static assets
- `test/`: Widget and app tests

## Current Screens

- `app_entry_page.dart`
- `login_page.dart`
- `map_home_page.dart`

## Getting Started

1. Install Flutter.
2. Run dependency install:

```bash
flutter pub get
```

3. Start the app:

```bash
flutter run
```
