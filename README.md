# POLAPP

POLAPP is a Flutter-based team project for building a police-related application.

## Project Structure

- `lib/main.dart`: App entry point
- `lib/pages/`: Main UI screens
- `lib/services/`: Shared business logic and service layer
- `assets/`: Images and other static assets
- `test/`: Widget and app tests

## Current Screens

- `app_entry_page.dart`
- `login_page.dart`
- `map_home_page.dart`

## Getting Started

1. Install Flutter.
2. Run dependency install:

```bash
flutter pub get
```

3. Start the app:

```bash
flutter run
```
